## 분석 페이지 (`index.html`)

### 버그 수정

| 항목 | 내용 |
|------|------|
| 국공립 분류 오류 | 원시 데이터의 `설립구분` 값이 `'국립'`·`'공립'`·`'국립대법인'` 등으로 저장되어 `_baseFilter`에서 전부 걸러지던 문제 수정. 결과적으로 전국 평균과 전국 사립 평균이 동일하게 표시되던 현상 해결 |
| 추이 분석 계산 키 불일치 | `manifest.json`의 `sort_key`(`"재학생 중도탈락률(%)"`)가 `calc_rules.json` 실제 키(`"중도탈락률(재학생)"`)와 달라 추이 분석에서 중도탈락률이 표시되지 않던 오류 수정 |

### 순위 보기 개선

- **순위 컬럼 정렬**: 순위(`_rank`) 컬럼을 클릭 가능한 정렬 컬럼으로 추가. 기본 정렬을 순위 오름차순(1위→꼴찌)으로 변경
- **특별법·기타 제외 토글**: 필터 바 지역 선택 우측에 토글 스위치 추가. 특별법국립·특별법법인·기타 설립구분 대학을 순위에서 제외 (기본값: 제외)
- **소수점 자리 수 통일**: 모든 비율 표시를 소수점 2자리로 통일
- **테이블 가로 스크롤**: `min-width: 640px` + `white-space: nowrap` 적용. 모바일에서 셀 텍스트가 세로로 줄바꿈되던 문제 해결, 좌우 스크롤로 전환

### 추이 분석 개선

- **그룹 평균 계산 기준 정비**: `_baseFilter`에서 특별법국립·특별법법인 제외. 허용 국공립 계열: `국공립`, `국립`, `공립`, `국립대법인`
- **사이드 패널 순서 변경**: 비교 그룹 → 대학 추가 비교 → Y축 범위 → 연도 선택
- **연도 선택 기본값**: 전체 연도 선택 → 최근 5년만 선택
- **연도 선택 표시**: 오름차순 → 내림차순, 최대 높이 168px + 스크롤, 2열 그리드 레이아웃으로 공간 효율화
- **Y축 범위 Auto 버튼**: 현재 시리즈 데이터의 min/max를 기반으로 여유 있는 범위를 자동 계산하여 적용. 항목 변경 시 자동 재계산 (기본값)
- **사이드 패널 높이**: `align-items: stretch` 적용으로 사이드 패널 하단이 오른쪽 콘텐츠(차트+표) 하단과 일치
- **소수점 자리 수 통일**: 툴팁·차트 레이블·Y축 눈금·요약 테이블 모두 소수점 2자리로 통일

### 모바일 대응

- **모바일 네비게이션**: 767px 이하에서 사이드바 대신 상단 탭 바(순위 보기 / 추이 분석) 표시
- **뷰 전환 동기화**: `switchView()` 호출 시 사이드바·모바일 탭 바 둘 다 `active` 상태 동기화

### 기타

- **그룹 비교·직접 비교 메뉴 제거**: 구현 계획 없이 `disabled` 상태로만 존재하던 nav 항목 및 관련 CSS(`.nav-item.disabled`, `.nav-badge`) 삭제
